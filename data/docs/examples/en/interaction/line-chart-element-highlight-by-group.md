# Element Highlight by Group

By configuring the `element-highlight-by-group` interaction type, the selected lines and points are highlighted, while other series elements are in a blurred state.

## Key Configuration

- `interactions` set the interaction of the series
- `point.state.blur` set the style for the blurred state of the point graphic element
- `area.state.highlight` set the style for the highlighted state
- `area.state.blur` set the style for the blurred state
